# docker-cpp-ci

A set of cxx build tools shipped in docker container mainly for CI.

Personal use.


## Base image

ubuntu:jammy

## Tools

Utilities

- git
- curl
- tar
- unzip

Build tools

- gcc
- g++
- clang
- cmake
- autoconf
- ninja
- llvm
- nasm

Test tools

- googletest

Happy building & tesing!
